# The-useful-github-resources

## 📔 這是甚麼?

我在 Github 上瀏覽的時候常常見到有用的資源，遇到這些東西時通常會把它 Star 或是 fork。

但是時間一久總會累積很多目錄，不少目錄還沒仔細研究過就忘記了。

所以我打算把他們匯集成一個備忘錄，裡面會寫入我認為具參考價值的目錄。

## 🥴 建構中

這項企畫還是寶寶階段，所以架構與規畫還沒有這麼完善。如果你有想推薦的項目或是建議歡迎提出。

---
